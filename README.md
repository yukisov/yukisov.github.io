Web site resources for yukisov.github.io
=====

How to Update articles
----------------------

1. Edit some files
2. $ jekyll serve
    - Build files
    - Run Web Server on local

jekyll コマンド
---------------

ウェブサーバ起動

```
jekyll server
```

ウェブサーバ起動（自動ビルドつき）

```
jekyll serve --watch
```

各HTML(or Markdown)ファイルを_siteフォルダに反映させる

```
jekyll build
```
